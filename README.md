# Digital Business Profile — Alessandro Stazzone

Personal digital business profile page — a bridge between Business, Digital and Technology in the omnichannel ecosystem.

## About

This repository hosts a standalone web profile presenting my role as Digital Business Specialist in the omnichannel ecosystem of Valentino.

The page is structured around two axes:

- Ownership areas — the value levers I preside along the customer journey (Customer & CRM, Order & Fulfillment, Digital Storefront, Content & Localization, Payments, Omnichannel Services, Analytics & VoC).
- Operating modes — how I generate value: Project & Design (strategic initiatives) and Operation & Continuous Improvement (business as usual, AMS, partner coordination).

A dedicated Cross-functional PM & Integration section highlights the transversal layer that connects all domains — the multiplier of the profile.

## Design system

- Palette: whites, greys, blacks with a violet accent (#6b3fa0)
- Typography: Oswald (industrial, uppercase) for headings, Inter for body copy
- Layout: minimal, card-based grid, dark highlight block for the transversal section

## Tech stack

- Plain HTML5 + CSS3 (no frameworks, no build step)
- Google Fonts (Oswald + Inter)
- Fully responsive via CSS Grid and clamp() typography
